# Dog Breed Identification (37 Classes)
Advanced Image Classification using Deep Learning to identify specific dog breeds.

### Project Overview
Going beyond simple binary classification, this model can distinguish between 37 different breeds using the Oxford-IIIT Pet Dataset.

### Key Features
- **Transfer Learning:** Leveraging powerful pre-trained models via **TensorFlow**.
- **Data Augmentation:** Increasing model robustness against different lighting and angles.
- **Multiclass Classification:** Softmax activation for 37 distinct categories.

### Tech Stack
- TensorFlow, Keras, NumPy, OpenCV.
